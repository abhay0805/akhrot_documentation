<Note>
  **Akhrot is a shared memory and context layer for multi-LLM systems**, powered by GML (Gigizs Multi-LLM Layer).
</Note>

Modern AI applications use multiple models — GPT, Claude, Gemini, Grok, and a growing ecosystem of open-source models. These models **do not share memory**. Every handoff loses context. Every session starts cold. Valuable information is repeatedly re-sent.

Akhrot solves this by providing **persistent, queryable memory** that survives:

- Model switches
- Agent handoffs
- Session restarts
- Context window limitations

## What you can build

- **Persistent chatbots** that remember every user across sessions and models.
- **Multi-agent systems** where a planning agent's decisions flow into a coding agent's execution.
- **Research assistants** that build durable knowledge over weeks of investigation.
- **Team knowledge bases** that capture decisions, rationale, and tribal knowledge.
