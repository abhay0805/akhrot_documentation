The Memory API is the core of Akhrot. It exposes ingest, recall, search, graph, and trace primitives.

## Base URL

```text
https://api.gml-akhrot.com/v1
```

## Authentication

All requests require a bearer token:

```text
Authorization: Bearer ak_live_...
```

## Errors

| Code | Meaning |
| --- | --- |
| `401` | Missing or invalid API key |
| `403` | Scope or quota denied |
| `404` | Memory not found |
| `422` | Validation error |
| `429` | Rate limited |
| `5xx` | Server error |

## Rate limits
