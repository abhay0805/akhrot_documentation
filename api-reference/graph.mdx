## Endpoint

```text
POST /v1/graph/query
```

## Request body

```json
{
  "userId": "user_42",
  "scope": ["project:billing"],
  "start": { "entity": "billing-service" },
  "traverse": ["DEPENDS_ON", "DECIDED", "SUPERSEDES"],
  "depth": 2,
  "limit": 200
}
```

| Field | Description |
| --- | --- |
| `start` | Seed node — `{ entity }`, `{ memoryId }`, or `{ tag }` |
| `traverse` | Edge types to follow |
| `depth` | Max hops |
| `limit` | Max nodes returned |

## Response

```json
{
  "nodes": [
    { "id": "e:billing-service", "type": "Entity", "label": "billing-service" },
    { "id": "mem_01HX...", "type": "Decision", "fact": "Use Postgres for billing" }
  ],
  "edges": [
    { "from": "mem_01HX...", "to": "e:billing-service", "type": "DEPENDS_ON" }
  ]
}
```
