The Akhrot dashboard renders your [memory graph](/core-concepts/memory-graph) as an interactive map. You can also pull the graph programmatically and render it in your own UI.

## In the dashboard

- Filter by user, team, project, time range
- Click a node to see source conversations and provenance
- Hover an edge to see its type and the facts that imply it
- Pin nodes, collapse clusters, export to PNG / JSON / GraphML

## Programmatic access

```ts
const graph = await akhrot.graph.query({
  userId: "user_42",
  start: { entity: "billing-service" },
  traverse: ["DEPENDS_ON", "DECIDED"],
  depth: 2,
});
```

## Render in React

```tsx
import { MemoryGraph } from "@akhrot/react";

export function Page() {
  return <MemoryGraph userId="user_42" rootEntity="billing-service" depth={2} />;
}
```
