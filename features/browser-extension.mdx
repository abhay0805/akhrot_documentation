## What it does

- **Auto-capture** — highlights and saves durable facts from your ChatGPT, Claude, Gemini, and Grok sessions.
- **Auto-recall** — silently injects relevant memories into the prompt area before you hit send.
- **One-click save** — right-click any message to save it as memory.
- **Cross-tool recall** — a memory written from a Claude tab shows up the next time you open ChatGPT.

## Privacy and scope

- Memories are scoped to **your account**, not the host site.
- The extension never sends raw page content to third parties.
- You can pause capture per-domain or globally from the toolbar.

<Note>
  The extension talks to the same Akhrot API the SDKs do. Anything captured is queryable from your code.
</Note>
