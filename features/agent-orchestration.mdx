In multi-agent systems, every handoff is a chance to lose context. Akhrot's orchestration primitives make handoffs lossless.

## The handoff problem

A planner agent decides _"build a billing dashboard with Stripe and Postgres."_ The executor agent receives only the task name. The rationale, alternatives considered, and user constraints are gone.

## Akhrot orchestration

```ts
await akhrot.orchestrate.handoff({
  fromAgent: "planner",
  toAgent: "executor",
  userId: "user_42",
  task: "Build billing dashboard",
  carryFacts: ["decision", "constraint", "preference"],
});
```

The `toAgent` automatically receives:

- All relevant **decisions** the planner made
- All **user constraints** in scope
- A trace ID for later audit

## Routing context to specific models

```ts
const slice = await akhrot.orchestrate.route({
  userId: "user_42",
  query: "draft the migration plan",
  targetModel: "claude-3.5-sonnet",
  budget: { tokens: 2000 },
});

const response = await anthropic.messages.create({
  model: "claude-3.5-sonnet",
  system: slice.system,
  messages: [...history, { role: "user", content: query }],
});
```
